Summary: Latitude vs. Cloudiness

We choose one weather element: "Cloudiness" to show how it would changes as examing location getting closer to the equator. Using Matplotlib to plot out the cloudiness data vs. latitude visualizations. In conclusion, we can clearly see that there is no correlation or any trend between cloudiness and latitude that can be drawn from this analysis.
